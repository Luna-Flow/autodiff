# ドキュメント標準

`Luna-Flow/autodiff` のドキュメントは、現在のブランチに存在する実装を記述し、公開 MoonBit API、代数的制約、リリース範囲と一致している必要があります。

## ルール

- 対応する各ロケールに `README.md`、`doc_standard.md`、サブシステム文書を用意する。
- 現在のブランチに実装済みの API と挙動のみを記述する。
- 用語は `luna-generic` と `arithmetic` の Luna Flow 名称に合わせる。
- `Dual[T]` を体または順序付きスカラーとして記述しない。
- 将来作業は明確に将来作業として示す。
- 各サブシステムディレクトリには `api.md`、`tutorial.md`、`design.md` を置く。
